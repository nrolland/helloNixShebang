A collection of self-contained scripts: the same "hello" payload (print a
table, say hi, fetch a page) re-expressed under a different self-sufficient
header each time — a shebang, a script-mode block, a PEP 723 comment. Same
payload, variable header; the weekly CI cron is the proof that every
mechanism below still resolves its dependencies and runs, not just that it
once did.

Most of the original scripts use Nix, either the classic `nix-shell`
shebang or the native `nix` (flake) shebang. For background:

- http://chriswarbo.net/projects/nixos/nix_shell_shebangs.html
- the ["Use as a interpreter"](https://nix.dev/manual/nix/stable/command-ref/nix-shell.html#use-as-a--interpreter)
  section of the Nix manual

Alongside Nix, the matrix below covers other self-contained mechanisms:
Haskell via `stack` and `cabal` script mode, Python via `uv` (PEP 723),
TypeScript via `deno`, and Clojure via `babashka`.
